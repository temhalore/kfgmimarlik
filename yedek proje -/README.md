# KFG Mimarlık Kurumsal Web Sitesi

Bu proje, dinamik component tabanlı, SEO uyumlu, çok dilli ve kolay yönetilebilir bir mimarlık sitesi için hazırlanmıştır.

- Tüm içerik ve görseller JSON/panel ile yönetilir.
- Renk ve font teması merkezi dosyadan değiştirilebilir.
- Bootstrap tabanlı responsive yapı.
- Tüm sayfa ve componentler SEO uyumlu.
- Panel ile içerik ve component yönetimi.

Ayrıntılı plan ve mimari için proje-plani.md dosyasına bakınız.
