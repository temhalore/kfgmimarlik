# Core Yapısı

Bu klasör, tüm projelerde ortak kullanılacak temel bileşenleri, fonksiyonları ve dosyaları içerir.

## İçerik
- header.js: Dinamik header fonksiyonları
- footer.js: Dinamik footer fonksiyonları
- content.js: Sayfa içeriklerini dinamik yükleyen fonksiyonlar
- i18n.js: Çoklu dil desteği fonksiyonları
- ortak.js: Ortak yardımcı fonksiyonlar
- index.js: Ana başlatıcı dosya
- index.html: Temel HTML şablonu
- ortak.css: Ortak stiller

Bu yapı, tema giydirme işlemlerinde referans alınacaktır.
